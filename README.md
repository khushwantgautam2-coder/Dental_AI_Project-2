# 🦷 PAI Diagnostic Assistant (YOLOv11 Nano)

An Explainable AI (XAI) dashboard for Periapical Index (PAI) scoring from dental X-rays.

## 🚀 Key Features
- **Triple-View Analysis:** Original, YOLO Bounding Boxes, and Heatmap Reasoning.
- **Explainable AI:** Uses heatmap visualization to show clinical focus areas.
- **Edge-Ready:** Powered by YOLOv11n (5.3MB) for real-time performance.
- **Clinical History:** Tracks and exports diagnostic logs to CSV.

## 🛠️ Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Dental_AI_Project.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
